#Codurance Web Site

##Installing if you like ruby

prereqs:
- install rvm https://rvm.io/
- `rvm install ruby`
- `rvm use ruby`
- `gem install bundler`
- `bundle install`

run locally:
- `bundle exec rake servequick`

In case of other problems, refer to the [troubleshooting section](#troubleshooting).
